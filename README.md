# HIT课程表（安卓）

仅支持 **哈尔滨工业大学（本部）教务导出的 `.xls` 网格课表格式**。

## 已实现功能

- 导入 `.xls` 课程表文件（系统文件选择器）
- 解析哈工大网格课表（按“星期列 × 节次行”）
- App 内课表表格视图（周一到周日、横纵双向滚动）
- 根据“第一周日期”自动按当前周显示课程
- App 内支持上一周 / 下一周 / 回到本周切换，方便查看不同周课表
- 课程格仅显示课程名预览，点击弹出居中详情卡（含缩放动画）
- 顶部紧凑图标操作栏（导入 / 设周 / 清空 / 操作指南）
- 桌面小组件显示“今日课程”，并对课程名做彩色区分
- 沉浸式显示（默认隐藏状态栏）

## 使用方式

1. 用 Android Studio 打开本目录。
2. 同步 Gradle 并运行到安卓设备。
3. 打开 App，点击“导入课表”，选择哈工大导出的 `.xls` 文件。
4. 点击日历图标设置“第一周日期”，即可按当前周过滤课程。
5. 使用课表上方的左右箭头切换周次；点击日历样式按钮回到本周。
6. 点击课程格可查看课程完整信息；点击“?”图标可查看使用说明。
7. 回到桌面，长按空白区域添加小组件，选择 **HIT课程表**。

## 安装提醒

如果新版 APK 无法直接覆盖安装旧版，请先在手机上卸载旧版 **HIT课程表**，再安装新的 APK。

## 解析规则（哈工大本部专用）

- 识别含“星期一~星期日”的表头行。
- 将“节次”列作为时间段（如“第1,2节”）。
- 课程单元格按换行拆分，按“课程名 + 详情行”配对。
- 详情行按 `教师[周次]地点` 提取教师、周次、地点。

> 非哈工大本部格式（或不是该导出模板）可能导入失败。
